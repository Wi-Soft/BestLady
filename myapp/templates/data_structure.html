Data Structure & Algorithms

Data Structure

A data structure is a way to store data.

Different types of data structures:

Primitive data structures are basic data structures provided by programming languages to represent single values, such as integers, floating-point numbers, characters, and booleans.

Abstract data structures are higher-level data structures that are built using primitive data types and provide more complex and specialized operations. Some common examples include arrays, linked lists, stacks, queues, trees, and graphs.
